## 📖 题型快速参考

| 题型 | 答案格式 | 示例 |
|------|---------|------|
| 填空题 | 字符串 | `"答案内容"` |
| 单选题 | 整数索引 | `0` (第一个选项) |
| 多选题 | 整数数组 | `[0, 2]` |
| 判断题 | 整数索引 | `0` (对) 或 `1` (错) |
| 简答题 | 字符串 | `"详细回答..."` |
| 矩阵填空 | 字典 | `{"q2_0": "答案1", "q2_1": "答案2"}` |
| 多项简答 | 字典 | `{"q12_0": "回答1", "q12_1": "回答2"}` |
| 下拉选择 | 整数索引 | `0` |
| 多项填空 | 字典或列表 | `["答案1", "答案2"]` |
| 级联下拉 | 字符串 | `"北京市-朝阳区"` |

详细说明请查看 [题型使用指南](QUESTION_TYPES_GUIDE.md)。
